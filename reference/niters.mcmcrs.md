# Number of Iterations

Gets the number of iterations (in a chain) of an MCMC object.

## Usage

``` r
# S3 method for class 'mcmcrs'
niters(x, ...)
```

## Arguments

- x:

  An object.

- ...:

  Other arguments passed to methods.

## Value

An integer scalar of the number of iterations.
