# Number of Terms

Gets the number of terms of an MCMC object.

## Usage

``` r
# S3 method for class 'mcmcarray'
nterms(x, ...)
```

## Arguments

- x:

  An object.

- ...:

  Other arguments passed to methods.

## Value

A integer scalar of the number of terms.
