# Is mcmcr object

Tests whether an object is an
[`mcmcr-object()`](https://poissonconsulting.github.io/mcmcr/reference/mcmcr-object.md).

## Usage

``` r
is.mcmcr(x)
```

## Arguments

- x:

  The object to test.

## Value

A flag indicating whether the test was positive.

## See also

Other is:
[`is.mcarray()`](https://poissonconsulting.github.io/mcmcr/reference/is.mcarray.md),
[`is.mcmcarray()`](https://poissonconsulting.github.io/mcmcr/reference/is.mcmcarray.md),
[`is.mcmcrs()`](https://poissonconsulting.github.io/mcmcr/reference/is.mcmcrs.md)

## Examples

``` r
is.mcmcr(mcmcr_example)
#> [1] TRUE
```
