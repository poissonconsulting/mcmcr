# An example mcmcr object

An example
[`mcmcr-object()`](https://poissonconsulting.github.io/mcmcr/reference/mcmcr-object.md)
derived from
[`coda::line()`](https://rdrr.io/pkg/coda/man/linepost.html).

## Usage

``` r
mcmcr_example
```

## Format

An object of class `mcmcr` of length 3.

## Examples

``` r
mcmcr_example
#> $alpha
#> [1] 3.718025 4.718025
#> 
#> nchains:  2 
#> niters:  400 
#> 
#> $beta
#>           [,1]     [,2]
#> [1,] 0.9716535 1.971654
#> [2,] 1.9716535 2.971654
#> 
#> nchains:  2 
#> niters:  400 
#> 
#> $sigma
#> [1] 0.7911975
#> 
#> nchains:  2 
#> niters:  400 
#> 
```
