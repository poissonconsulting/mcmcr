# Number of Chains

Gets the number of chains of an MCMC object.

## Usage

``` r
# S3 method for class 'matrix'
nchains(x, ...)
```

## Arguments

- x:

  An object.

- ...:

  Other arguments passed to methods.

## Value

An integer scalar of the number of chains.
